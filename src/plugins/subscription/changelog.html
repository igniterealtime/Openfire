Subscription Plugin Changelog

1.2.2 -- February 02, 2010

  • Updated license from GPL to Apache.

1.2.0 -- April 12, 2007

  • Presence handling by the server was being skiped.
  • Now requires Openfire 3.3.0.

1.1.0 -- February 06, 2007

  • Now requires Openfire 3.2.0

1.0.1 -- October 06, 2006

  • Updated to use compression offered by Openfire 3.1

1.0.0 -- July 17, 2006

  • Initial release.

1.0.0 - Beta 4 -- June 9, 2006

  • Added the ability to specify a "white list" of users who can still manage their own subscriptions.

1.0.0 - Beta 3 -- April, 18 2006

  • Added the ability to automatically reject all subscription requests.

1.0.0 - Beta 2 -- February, 17 2006

  • Added the ability to set subscription service properties via the Admin Console.
  • Fixed potential NullPointerException in packet interceptor.

1.0.0 - Beta 1 -- February, 16 2006

  • Initial beta release.

Client Control Plugin Changelog

2.1.3 -- July 30, 2017

  • [OF-1354] - Add option to Client Control plugin to disable Start a chat in Spark.
  • This update corresponds with ticket SPARK-1946 and requires Spark version 2.9.0 and above for new functionality.

2.1.2 -- May 8, 2017

  • [OF-1325] - Adds 'History Transcripts' and fixes 'History Settings' in Client Control.
  • NOTE: This update corresponds to ticket SPARK-1915 and requires Spark version 2.9.0 and above for proper functionality.
  • If a previous version of this plugin was installed, the previous History Settings will be carried over and split into the new settings.
  • History Settings - Corresponds to HIDE_HISTORY_SETTINGS in 'default.properties' file and performs the following actions when set to Disabled:
    • Removes history settings from Preferences menu
    • Removes 'clear' option from context menu in the chat window

  • History Transcripts - Corresponds to HISTORY_DISABLED in 'default.properties' file and performs the following actions when set to Disabled:
    • Removes history settings from Preferences menu
    • Removes 'clear' option from context menu in the chat window
    • Removes history button from the chat window
    • Removes 'View Log' option from context menu in chat window
    • Removes history from the chat window
    • Removes 'View contact history' option from contact's context menu
    • Does not save history into transcript files
  • [OF-1277] - Change setting name to 'Invisible Login and Status'

2.1.1 -- December 30, 2016

  • [OF-1258] - Adds an option to enable/disable anonymous login in Spark.
  • This update corresponds with ticket SPARK-1852 and requires Spark version 2.8.3 and above for new functionality.
  • [OF-1260] - Improve CSRF error message.

2.1.0 -- November 24, 2016

  • [OF-1221] - Adds many more options from 'default.properties' which now allow administrators to enable them network wide.
  • This update corresponds with ticket SPARK-1835 and requires Spark version 2.8.2 and above for functionality.
  • Locale files updated are: EN, cs_CZ, ES, FR, PT_BR, and zh_CN.

2.0.1 -- November 4, 2016

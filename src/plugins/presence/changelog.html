Presence Plugin Changelog

1.4.0 -- March 29, 2007

  • Now requires Openfire 3.3.0.
  • The administrator can now set custom values for the presence text.
  • Updated default presence images.
  • Added URL information to admin console page.

1.3.3 -- November 03, 2006

  • Fixed NPE when no target JID was found in HTTP request.

1.3.2 -- October 06, 2006

  • Updated to use compression offered by Openfire 3.1

1.3.1 -- July 19, 2006

  • Added support for retrieving the user's presence status as plain text.

1.3.0 -- July 10, 2006

  • Added support for probing presence of components.
  • Fixed minor problem in readme.html -- the example Apache redirect was incorrect.

1.2.0 -- April 19, 2006

  • Added documentation for configuring the plugin to work well with firewalls.
  • [JM-602] - Presence plugin now handles HTTP redirection. (1 vote)

1.1.2 -- March 9, 2006

  • Now requires Openfire 2.5.1

1.1.1 -- December 15, 2005

  • Now requires Openfire 2.4.0

1.1 -- Jul 25, 2005

  • Large updates to documentation and protocol.

1.0 -- Jul 20, 2005

  • Initial release.

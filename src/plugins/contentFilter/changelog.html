ContentFilter Plugin Changelog

1.5.0 -- March 29, 2007

  • Now requires Openfire 3.3.0.

1.4.0 -- February 06, 2007

  • Now requires Openfire 3.2.0.

1.3.1 -- October 06, 2006

  • Updated to use compression offered by Openfire 3.1

1.3.0 -- May 8, 2006

  • Added ability to allow messages even when content has matched (thanks to Eduardo for the feature request)
  • Added ability to restore plugin to its factory defaults.

1.2.2 -- January 26, 2006

  • Invalid pattern was preventing the plugin from being loaded.

1.2.1 -- January 09, 2006

  • Minor patch to improve handing of email notification failures.

1.2 -- Jan, 2006

  • Added support to notify administrator by email on content match notification.
  • Added support to include original packet when sending violation notifications.
  • Added support to filter users presence status.
  • Increased size of patterns input area to allow more patterns.
  • Fixed masking function.
  • Added debug logging to help users identify problems.
  • Updated readme.html

1.0.1 -- December 15, 2005

  • Now requires Openfire 2.4.0

1.0 -- July, 2005

  • Initial release.

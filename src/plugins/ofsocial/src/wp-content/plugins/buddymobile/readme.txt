== Description ==
Mobile plugin for optimized mobile experience on BuddyPress sites.

== Installation ==

= Automatic Installation =

1. From inside your WordPress administration panel, visit 'Plugins -> Add New'
2. Search for `BuddyMobile` and find this plugin in the results
3. Click 'Install'
4. Activate the plugin through the 'Plugins' menu in WordPress

= Manual Installation =

1. Upload `buddymobile` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

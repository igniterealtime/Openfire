Openfire Meetings Plugin Changelog

0.3.1 -- December 12, 2015

  • Fixed issue with broken presence broadcast
  • Fixed issue with broken desktop/screen sharing
  • Fixed issue with missing jnsctp.dll dependencies
  • Removed some more redundant or duplicate jar files in jitsivideobridge lib

0.3.0 -- October 12, 2015

  • [OF-953] - Updated JSP libraries.
  • Requires Openfire 3.11.0.

0.2.3 -- Nov 14th, 2015

  • Simplified build, removed classes folder in WEB-INF
  • Changed default value for voip gateway to false
  • Removed some redundant and duplicate jar file in jitsivideobridge lib
  • Removed tinder.jar and whack.jar which caused ComponentManagerFactory.getComponentManager() to fail with OF 3.11

0.2.2 -- Oct 9th, 2015

  • Reverted Jitsi Videobridge latest code because of issues on linux builds

0.2.1 -- Oct 8th, 2015

  • Updated Jitsi Videobridge to latest code
  • Fixed issues with Jigasi voip gateway
  • Added Multicast RTP to Jigasi voip gateway

0.2.0 -- Aug 29th, 2015

  • Updated Jitsi Videobridge to latest code
  • Removed support for jingle nodes, redundant.
  • Used IP Address instead of host name for NAT_HARVESTER_LOCAL_ADDRESS and NAT_HARVESTER_PUBLIC_ADDRESS.

0.1.9 -- Aug 4th, 2015

  • Fixed issue in build process (the J2EE version that is used should not be platform-dependant).

0.1.8 -- May 20th, 2015

  • Added conference UI customization to Meetings Settings Admin web page
  • Fixed issue with room locking not working
  • Updated Jitsi Videobridge

0.1.7 -- May 13th, 2015

  • Logs openfire meeting audit events with Openfire security auditor
  • Logs meeting SIP calls via SIP plugin
  • Added support for audio conferencing meetings (no video)
  • Updated Jitsi Videobridge

0.1.6 -- May 9th, 2015

  • Added support for media relaying using the JingleNodes plugin

0.1.5 -- May 4th, 2015

  • Implemented Meeting Planner and automatic invites. Requires bookmarks from the clientcontrol plugin
  • Fixed error modified setLocalDescription failed with desktop share
  • Prevents sending invalid presence packets, before MUC jid is defined
  • Added support for web+meet: protocol handling
  • Updated Jitsi Videobridge

0.1.4 -- March 13th, 2015

  • Added focus username and password to the admin settings page
  • Updated Jitsi Videobridge

0.1.3 -- Feb 28th, 2015

  • Implemented server-side Smack Virtual XMPP Connection for conference focus instead of default TCP connection
  • Updated chrome extension to 0.0.5
  • Updated Jitsi Videobridge

0.1.2 -- Feb 15th, 2015

  • Added shared scrum board (post-it notes) Openfire Meetings Collaboration Application
  • Added "disable cursor" feature to Openfire Meetings Collaboration API
  • Added "ofmeet.js" and "ofmeet.css" to enable Openfire Meetings to be used as an API in other web applications
  • Provided webcomponent to wrap Openfire Meetings API into an HTML tag <openfire-meetings>
  • Updated chrome extension to 0.0.3

0.1.1 -- Feb 8th, 2015

  • Added co-browsing to collaboration api
  • Moved collaboration api to chrome extension for co-browsing
  • Updated chrome extension to 0.0.2

0.1.0 -- Feb 4th, 2015

  • Implemented Openfire Meeting collaborative applications api
  • Added simple sample applications for rich text editing and whiteboard

0.0.9 -- Jan 26th, 2015

  • Fixed error when webcam is not available
  • Jitsi Videobridge server code refresh

0.0.8 -- Jan 16th, 2015

  • Fixed race condition causing focus xmpp connection to fail
  • Increased response timeout for focus
  • Improved error handling for read-only user providers like LDAP
  • Disable SIP gateway with a property (org.jitsi.videobridge.ofmeet.sip.enabled)

0.0.7 -- Jan 5th, 2015

  • Implemented PDF cursor sharing

0.0.6 -- Dec 29th, 2014

  • Fixed fault with exception when clientcontrol plugin is missing
  • Fixed fault with focus when hostname is localhost
  • Removed install logic for chrome extension. Always requires a manual install
  • Fixed broken audio and video bandwidth configuration with server-side focus change.

0.0.5 -- Dec 15th, 2014

  • Added support for clientcontrol plugin groupchat bookmarks and pdf urls
  • Added jicofo, server-side conference focus
  • Updated to latest Jitsi Meet and Jitsi Videobridge

0.0.3 -- Dec 6th, 2014

  • Changed version to 3.9.9 to overcome bug in openfire version control
  • Added recording parameters to admin web page
  • Updated Jitsi Meet and Jitsi Videobridge
  • Added Spark plugin
  • Added support for Openfire vCard avatars

0.0.2 -- Nov 30th, 2014

  • Added Candy and Fastpath.

0.0.1 -- Nov 27th, 2014

  • Initial release.

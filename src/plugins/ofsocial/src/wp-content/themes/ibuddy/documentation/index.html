Setting Pages

Setting the Home page:

  1. Go to Pages → Add New
  2. Give your page a title like Home
  3. Select the Home template from the template select option
  4. Hit publish

 

Now we have to to apply the Hom page we created as the front page for the website:

  1. Go to Settings → Reading
  2. Front page displays select Static page
  3. Front page select the Home page we created
  4. Hit Save changes

 

Now the Home page is set and ready. ( For further settings see the Home page Settings section )


 

Setting the Blog page:

  1. Go to Pages → Add New
  2. Give your page a title like Home
  3. Select the Home template from the template select option
  4. Hit publish

 

Now the Blog page is set and ready. ( Please note: you don't have to apply any changes under Settings → Reading as this page template will already retrieve you blog posts )

  1. Logo

Here where you can upload your own logo that will be placed in:

The header. It will be used instead of the texture logo (recommended max size H80px , w600px)

You can chose the skin color you prefer to use. If you don't chose any, than the default will be used

Pink

Orange

Green

 

  1. Google Analytics Code

    If you use Google Analytics to Track your website visitors than you can insert your tracking code in the text box so wit will be inserted before the closing of the </head> tag.

Advanced

 

  1. Lock-down BuddyPress

    This option will Lock-down your BuddyPress pages to only logged-in Members. So non logged-in Members/Visitors will not have any access and if they try to access one of BuddyPress pages they will redirected to the page you specify in the option below.

  • Redirect Page

    If you activated the option above than chose the page that will be used for the redirection

    for logged-out Members/Visitors. Please note: If you don't select any, than the login page will be the default


 

  1. Lock-down Home Page

    With this option you have the possibility to prevent access to the Home page by the logged-in Members. But anyway they can still see access it as long as they are logged-out.

    So, if this option is active and a member uses the login form on the Home page, than the Member will be redirected to the profile page after a successful login.

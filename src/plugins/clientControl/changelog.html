Client Control Plugin Changelog

2.1.1 -- December 30, 2016

  • [OF-1258] - Adds an option to enable/disable anonymous login in Spark.
  • This update corresponds with ticket SPARK-1852 and requires Spark version 2.8.3 and above for new functionality.
  • [OF-1260] - Improve CSRF error message.

2.1.0 -- November 24, 2016

  • [OF-1221] - Adds many more options from 'default.properties' which now allow administrators to enable them network wide.
  • This update corresponds with ticket SPARK-1835 and requires Spark version 2.8.2 and above for functionality.
  • Locale files updated are: EN, cs_CZ, ES, FR, PT_BR, and zh_CN.

2.0.1 -- November 4, 2016

  • [OF-1211] - Updated description for 'Broadcasting' option to correspond with issue resolved in ticket SPARK-1827. Locale files updated are EN, CZ, ES, FR, PT_BR.

2.0.0 -- January 11, 2016

  • No longer supports Bookmarks functionality (this functionality has been moved to the new Bookmarks Plugin).

1.3.1 -- Nov 18, 2015

  • [OF-953] - Updated JSP libraries.

1.3.1 -- Nov 08, 2015

  • Added Name as a nickname option to Group Chat Bookmark.

1.3.0 -- June 22, 2015

  • Requires Openfire 3.11.0.

1.2.0 -- September 13, 2013

  • Requires Openfire 3.9.0.

1.1.0 -- December 1, 2009

  • Now requires Openfire 3.7.0.
  • Applied Java generics.
  • [OF-53] - Replace custom logging implementation with a third party library.

1.0.3 -- August 27, 2008

  • Upgraded for Openfire 3.6.0 API.

1.0.2 -- June 20, 2008

  • Authentication is no longer required to download new Spark version.

1.0.1 -- May 26, 2008

  • Authentication is no longer required to download new Spark version.
  • Updated client name from Gaim to Pidgin.
  • Added missing icons.
  • Improved readme documentation.

1.0.0 -- May 13, 2008

  • Functionality ported from Openfire Enterprise, now Open Source.
